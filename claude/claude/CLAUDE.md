# Claude Skills Collection - プロジェクトコンテキスト

## プロジェクト概要

Claude Code のスキル、エージェント、ルール、コマンドを管理するコレクションリポジトリ。

## リポジトリ構成

- `skills/` - SKILL.md ベースのスキル集（カテゴリ別に整理）
- `agents/` - エージェント定義（.md ファイル）
- `rules/` - パス・トピック別ルール
- `commands/` - カスタムスラッシュコマンド
- `templates/` - 各種テンプレート
- `docs/` - ドキュメント・ガイド

## コーディング規約

### スキル作成時のルール

1. **1スキル1ディレクトリ**: 各スキルは独自のディレクトリを持ち、必ず `SKILL.md` を含める
2. **YAMLフロントマター必須**: `name` と `description` を必ず記載する
3. **参考ファイルは `references/` に配置**: 大きなドキュメントやサンプルは `references/` サブディレクトリに格納
4. **日本語で記述**: スキルの説明やドキュメントは日本語で統一

### エージェント作成時のルール

1. **役割を明確に定義**: 最初に役割（ロール）を明示する
2. **ツール制限を記載**: `disallowedTools` や `tools` でツールの使用範囲を制限
3. **出力フォーマットを指定**: 期待する出力の形式を明記

### ルール作成時のルール

1. **対象パスを明記**: どのファイルパスに適用されるか記載
2. **具体例を含める**: NG例とOK例を必ず含める

## 言語

- ドキュメント・スキル定義: 日本語
- コード例: 英語（変数名・関数名等）
